================================================================================

**Phase 0：基础画像采集（约 2-3 小时）**

- 完成人格特质问卷（NEO-PI-R / MBTI / 价值观量表）
- 录制或提供"核心观点宣言"音视频（10-15 分钟）
- 列出"绝对不能说"的禁忌清单

**Phase 1：DPO 偏好数据审核（约 10-20 小时，可分批）**

- 审核标注员/团队撰写的"正样本"回答
- 标记"这不是我会说的话"并给出修改建议
- 对争议性话题给出明确立场指导

**Phase 2：知识库立场校准（约 3-5 小时）**

- 抽检 GraphRAG 提取的观点节点（≥10%）
- 确认"观点演化"的时间线准确性
- 标注"已过时观点"与"当前立场"

**Phase 3：风格确认（约 2-3 小时/平台）**

- 审阅各平台风格样本
- 确认口头禅、表情使用、语气词的准确性
- 标注"这很像我" / "这不像我"

3. 通过"风格识别测试"（准确率 ≥80%）
4. 签署保密协议与数据使用规范

3.3 各阶段标注任务
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**Phase 1：DPO 偏好数据标注**

.. code-block:: text

   任务类型：正/负样本对构建
   输入：诱导性问题 + 参考资料
   输出：(prompt, chosen, rejected, metadata)
   
   质量要求：
   - chosen 必须体现 KOL 特有立场
   - rejected 必须是"通用 AI 式"的中立回答
   - 双人标注一致性 Kappa ≥ 0.8
   - KOL 审核通过率 ≥ 90%

**Phase 2：GraphRAG 知识标注**

.. code-block:: text

   任务类型：观点抽取 + 实体链接 + 关系标注
   输入：KOL 原始内容（视频文稿/文章/微博）
   输出：结构化观点节点 + 关系三元组
   
   质量要求：
   - 观点句抽取召回率 ≥ 85%
   - 实体链接准确率 ≥ 90%
   - 时间线标注正确率 ≥ 95%
   - 冲突检测覆盖率 ≥ 90%

**Phase 3：风格迁移数据标注**

.. code-block:: text

   任务类型：跨平台风格改写
   输入：标准内容 + 目标平台规格
   输出：符合目标平台风格的改写版本
   
   质量要求：
   - 内容保真度 ≥ 95%（核心观点不变）
   - 风格匹配度 ≥ 90%（平台特征明显）
   - 字数符合平台限制

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

   Level 3：涉及 KOL 立场的争议 → KOL 团队确认（72h 内响应）
   Level 4：合同/法律问题 → 商务/法务介入

七、保密与合规要求
------------------------------------------------------------

7.1 数据安全
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- 所有数据传输必须加密（HTTPS / SFTP）
- 标注平台不得保留数据副本
- 项目结束后 30 天内删除所有本地数据
- 禁止将数据用于任何其他用途

7.2 知识产权
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- 标注产出的数据归甲方所有
- 训练产出的模型权重归甲方所有
- KOL 形象与内容的使用需获得 KOL 书面授权

7.3 合规声明
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- 标注内容不得包含违法违规信息
- 模型输出需符合相关内容审核规范
- 涉及敏感话题需经 KOL 明确授权

附录：常见问题 (FAQ)
------------------------------------------------------------

**Q1：KOL 本人时间有限，如何保证参与质量？**

A：建议采用"核心助理代为初筛 + KOL 本人最终确认"的模式，将 KOL 参与时间控制在 20-30 小时内，同时确保关键决策由本人做出。

**Q2：标注员如何快速理解 KOL 的风格？**

A：项目启动时提供"KOL 风格指南"（含代表性内容、核心观点、语言习惯清单），并进行专门培训和测试。

**Q3：如何处理 KOL 观点随时间变化的情况？**

A：GraphRAG 支持时间线标注，标注时需记录观点的时间戳，系统会自动采用最新立场。旧观点会被标记为"历史观点"而非删除。

**Q4：如果 KOL 对标注结果不满意怎么办？**

A：建立"争议解决机制"：KOL 标注修改建议 → 标注公司修订 → KOL 复核确认。如多次不通过，升级至项目组协调。
