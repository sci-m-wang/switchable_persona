
Multi-Layer Role-Playing Framework
==================================

A Universal Framework for Digital Celebrity Simulation

本框架提供了一套完整的技术方案，用于构建能够保持人格一致性的数字名人模拟系统。
通过三层架构（特质层、知识层、风格层）的解耦设计，实现跨场景的适应性输出。
